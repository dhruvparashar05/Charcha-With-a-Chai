# ☕ Charcha With a Chai

**Charcha With a Chai** is a cozy AI-powered developer hangout where you can have natural conversations with two unique mentor personas. Whether you're debugging code, learning a new technology, preparing for interviews, or just discussing programming concepts, pull up a virtual chair and start a conversation over chai.

🔗 **Live Demo:** https://charcha-with-a-chai.onrender.com

---

## ✨ Features

- 🤖 Chat with two unique AI mentor personas
- 💬 Natural Hinglish conversations
- ⚡ Fast responses powered by **Google Gemini 2.5 Flash**
- 🎨 Modern and responsive UI
- 💭 Typing indicator
- 🔄 Seamless persona switching
- 📱 Mobile-friendly design
- ☕ Cozy chai-inspired theme

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/charcha-with-a-chai.git
cd charcha-with-a-chai
```

### 2. Install dependencies

```bash
npm install
```

### 3. Create a `.env` file

```env
API_KEY=YOUR_GEMINI_API_KEY
```

### 4. Start the server

```bash
npm start
```

Open your browser and visit:

```
http://localhost:3000
```

## 💡 Inspiration

The idea behind **Charcha With a Chai** is simple—

Programming discussions feel more enjoyable when they happen like a casual conversation over a cup of chai. Instead of interacting with a generic chatbot, users can choose a mentor whose personality matches their learning style.

Whether you prefer calm guidance or witty banter, there's always someone ready for a tech discussion.
